## 🚀 Project Overview

This project is a **Sales Analytics Dashboard** built using Python and data visualization tools.
It helps analyze sales performance across different categories, regions, and time periods.

The goal of this project is to **extract insights from raw sales data** and present them using interactive and meaningful visualizations.

---

## 🎯 Key Features

* 📈 Sales analysis by Category
* 🌍 Region-wise sales performance
* 📅 Monthly sales trends
* 🏆 Top-performing products
* 💰 KPI Metrics:

  * Total Sales
  * Total Profit
  * Total Orders
* 🎛️ Interactive Filters (Slicers):

  * Category
  * Region
  * Date

## ⚙️ How to Run This Project

### Step 1: Clone the Repository

```
git clone https://github.com/YOUR-USERNAME/sales-analytics-project.git
cd sales-analytics-project
```

### Step 2: Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate
```

### Step 3: Install Dependencies

```
pip install -r requirements.txt
```

### Step 4: Run Notebook

```
jupyter notebook
```
